# Personal Portfolio Website

This is my **Final Project for IT Elective 4**.  
It is a responsive personal portfolio website designed to showcase my profile, technical skills, and academic projects.

The website serves as my digital resume and demonstrates my understanding of **HTML5 structure, CSS3 styling, responsiveness, and GitHub deployment**.

---

---

## Tech Stack
- HTML5 (Semantic Structure)
- CSS3 (Custom Styling, Flexbox, Grid)
- Git & GitHub
- GitHub Pages (Deployment)

---

## Features
- Fully responsive design (mobile, tablet, desktop)
- Semantic HTML5 structure
- Hero section with call-to-action
- About Me section with profile introduction
- Skills section with animated progress bars
- Projects gallery with clickable project cards
- Separate project detail pages
- Contact section with basic contact information
- Clean, modern UI design
