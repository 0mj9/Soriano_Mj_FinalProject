LibraFlow System

RFID-Based Library Management

Project Overview

This system automates library operations using RFID technology for faster book borrowing and real-time inventory tracking.

System Architecture

The system uses a web-based architecture where users connect via the Internet to the LibraFlow application, which communicates with an RFID Scanner API and a central database.

Key Features

  • RFID-based borrowing and returning
  • Real-time book availability tracking
  • Automated due-date notifications
  • Role-based access for admin and users
